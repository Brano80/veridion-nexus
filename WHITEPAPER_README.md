# Veridion Nexus Investment Whitepaper

## Overview

This directory contains a comprehensive investment whitepaper for Veridion Nexus, designed for potential investors, partners, and stakeholders.

## Files

- **VERIDION_NEXUS_WHITEPAPER.md** - Complete whitepaper in Markdown format
- **generate-whitepaper-pdf.ps1** - PowerShell script to convert to PDF (Windows)
- **generate-whitepaper-pdf.sh** - Bash script to convert to PDF (Linux/Mac)

## Converting to PDF

### Option 1: Using Pandoc (Recommended)

**Windows:**
```powershell
.\generate-whitepaper-pdf.ps1
```

**Linux/Mac:**
```bash
chmod +x generate-whitepaper-pdf.sh
./generate-whitepaper-pdf.sh
```

**Prerequisites:**
- Install [Pandoc](https://pandoc.org/installing.html)
- Install a PDF engine (wkhtmltopdf, xelatex, or pdflatex)

### Option 2: Online Converters

If you don't have Pandoc installed, you can use online converters:

1. **Markdown to PDF**: https://www.markdowntopdf.com/
   - Upload `VERIDION_NEXUS_WHITEPAPER.md`
   - Download PDF

2. **Dillinger**: https://dillinger.io/
   - Open the markdown file
   - Click "Export as" → "PDF"

3. **VS Code Extension**: 
   - Install "Markdown PDF" extension
   - Right-click on `.md` file → "Markdown PDF: Export (pdf)"

### Option 3: Manual Conversion

1. Open the markdown file in a Markdown editor (Typora, Mark Text, etc.)
2. Export/Print as PDF
3. Adjust formatting as needed
